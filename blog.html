Question-1: What are the different ways to select an element in the DOM?

Different ways are- 1.by ID,, 2.by class Name, 3.by Tag Name, 4.by Attribute, 5.by Elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets/sets HTML Content. innerText: Gets/sets visible Content. textContent: Gets/sets all text.

Question-3: What is event delegation in the DOM?

Event Delegation is technique where you add an event listener to a parent element instead of indibidual child elements.

Question-4: What is event bubbling in the DOM?

It is a mechanism where an event starts from the target element and propagates up to its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

In javascript we can create by using createElement("tag"), add by using appendChild(), and remove by using removeChild().
